# 🎮 War Game - Simulador de Batalhas em C

Jogo de estratégia baseado no clássico War, desenvolvido em C para a disciplina de Estrutura de Dados. O projeto evolui em três níveis de complexidade, implementando conceitos fundamentais de programação.

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como trabalho prático da disciplina de Estrutura de Dados, demonstrando a aplicação de conceitos como:
- Ponteiros e alocação dinâmica de memória
- Structs e tipos abstratos de dados
- Modularização e boas práticas de programação
- Gerenciamento de memória

## 🎯 Níveis de Implementação

### Nível Básico - Cadastro de Territórios
- Cadastro de exércitos com nome, cor e tropas
- Exibição dos territórios cadastrados
- Estrutura básica de dados

### Nível Aventureiro - Sistema de Batalha
- Alocação dinâmica de territórios
- Sistema de ataque com rolagem de dados
- Atualização de territórios após batalhas
- Gerenciamento de memória

### Nível Mestre - Missões Estratégicas
- Sistema de missões aleatórias
- Múltiplos jogadores
- Condições de vitória variadas
- Verificação automática de objetivos

📝 Aprendizados
Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes de:

Alocação Dinâmica: Uso de malloc/calloc e free

Ponteiros: Manipulação direta de memória

Modularização: Separação de responsabilidades em funções

Passagem de Parâmetros: Por valor e por referência

Gerenciamento de Memória: Prevenção de vazamentos
